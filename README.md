SBDREC is a powerlifting record keeper.

## Requirements

Make sure you have Node and NPM installed:

```bash
node -v
# v20.6.1
```
```bash
npm -v
# 10.1.0
```

## Installation

Run the commands:
```bash
npm install
```
```bash
npm run dev
```
